# Cape-Stone-Project

This project is a basic online learning platform website created using only HTML5 and CSS3.
features-
1-  It includes a #header with navigation links
2-  A hero banner with a title and call to action button
3-  Course cards displayed in a responsive grid,testimonials and a contact form with proper labels.
4-  The layout is built using semantic HTML elements along with Flexbox and CSS Grid to make the page fully responsive.
5-  The design uses simple colors, fonts, hover effects and clean spacing to make the website easy to read and use.
